## Prerequisites
```bash
Node.js (v20.5.1)
Yarn
TypeScript
PostgreSQL Database Server
```

## Installation

```bash
$ cp .env.example .env
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
